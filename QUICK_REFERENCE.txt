## 📊 API Endpoints

```
POST /api/generate-layout
  Input: {population, temperature, weather, roads}
  Output: {layout_image, report}

POST /api/get-recommendations
  Input: {population, temperature, weather}
  Output: {density, amenities, road_width}

POST /api/contact
  Input: {name, email, subject, message}
  Output: {success, message}
```

## 💡 Planner Usage

1. Go to: http://localhost:5000/planner
2. Fill form:
   - Population (1,000 - 10M+)
   - Temperature (in °C)
   - Weather (Sunny, Rainy, etc.)
   - Roads (Basic, Normal, Advanced, Modern)
3. Click "Generate Layout"
4. See grid visualization + analysis

## ⚙️ Troubleshooting

| Problem | Solution |
|---------|----------|
| Port 5000 in use | Change port in app.py |
| Module not found | Run: `pip install Flask Flask-CORS numpy matplotlib scipy pandas` |
| Page not loading | Clear browser cache (Ctrl+Shift+Del) |
| Layout won't generate | Fill all form fields, check console (F12) |
