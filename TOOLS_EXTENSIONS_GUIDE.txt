## 🚀 How to Use These Tools

### Using Python Packages in Code
```python
from flask import Flask
import numpy as np
import matplotlib.pyplot as plt
from scipy import stats
import pandas as pd

app = Flask(__name__)  # Use Flask

@app.route('/api/data')
def get_data():
    data = np.array([1, 2, 3, 4, 5])  # Use NumPy
    df = pd.DataFrame(data)  # Use Pandas
    return df.to_json()  # Return JSON
```

### Using VS Code Features
1. **Debugging:** Press F5 or click Debug icon
2. **Code Runner:** Right-click → Run Code
3. **Live Server:** Right-click HTML → Open with Live Server
4. **Linting:** ESLint errors show as red underlines
5. **IntelliSense:** Ctrl+Space for completions

## 🔧 Configuration Files

### VS Code Settings (Recommended)
Create `.vscode/settings.json` in project root:
```json
{
  "python.linting.enabled": true,
  "python.linting.pylintEnabled": true,
  "python.formatting.provider": "black",
  "editor.formatOnSave": true,
  "javascript.validate.enable": true,
  "[python]": {
    "editor.defaultFormatter": "ms-python.python",
    "editor.formatOnSave": true
  }
}
```
