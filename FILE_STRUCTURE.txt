SETUP STEPS
===========
1. Navigate to: c:\visual studio\project1

2. Option A - Automatic (Windows):
   > run.bat

3. Option B - Automatic (Mac/Linux):
   $ chmod +x run.sh
   $ ./run.sh

4. Option C - Manual:
   $ cd backend
   $ python -m venv venv
   $ venv\Scripts\activate  (Windows)
   $ source venv/bin/activate (Mac/Linux)
   $ pip install -r requirements.txt
   $ python app.py

5. Visit: http://localhost:5000


API USAGE EXAMPLES
==================

Generate Layout:
POST /api/generate-layout
Content-Type: application/json

Request:
{
    "population": 150000,
    "temperature": 25,
    "weather": "Moderate",
    "roads": "Normal"
}

Response:
{
    "success": true,
    "layout_image": "data:image/png;base64,...",
    "report": {
        "population": "150,000",
        "density": "High",
        "temperature": "25°C",
        "weather": "Moderate",
        "road_width": "Width Level: 3/4",
        "amenities": ["Parks", "Shopping Malls", ...],
        "recommendations": ["Develop rapid transit", ...]
    }
}


GRID VISUALIZATION
==================
The planner generates a 10x10 grid with:

Zone Colors:
  Gray       = Roads & Transportation
  Yellow     = Residential
  Red        = Commercial
  Green      = Parks & Green Space
  Dark Gray  = Industrial

Zone Distribution:
  40% - Roads (main + secondary)
  25% - Residential
  15% - Commercial
  15% - Parks
  5%  - Industrial (if high density)

TROUBLESHOOTING
===============
Port 5000 in use:
  → Change port in app.py, line: app.run(port=5001)

Module not found:
  → Activate venv and run: pip install -r requirements.txt

Layout not generating:
  → Check browser console (F12) for errors
  → All form fields must be filled

Page not loading:
  → Clear browser cache (Ctrl+Shift+Delete)
  → Use private/incognito mode
