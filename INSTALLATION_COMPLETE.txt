## 🛠️ Troubleshooting Guide

### Issue: Port 5000 already in use
**Solution:** Edit `app.py` and change `app.run(port=5001)`

### Issue: Module not found
**Solution:** Run `pip install Flask Flask-CORS numpy matplotlib scipy pandas` again

### Issue: Layout not generating
**Solution:** Check browser console for errors, ensure all form fields are filled

### Issue: Flask import error
**Solution:** Use full Python path: `C:\Users\arivu\AppData\Local\Microsoft\WindowsApps\python3.13.exe app.py`
